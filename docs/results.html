As of May 6, 2026, the codebase is complete and the NYU-v2 export is still in progress. Performance metrics will populate after the first training run.

Paper story preserved

The project keeps the original presentation’s main structure: relative depth reasoning, Depth Anything Small, NYU-v2 evaluation, and an optional pseudo-label self-training extension.

Built for this machine

The implementation is sized for an RTX 3050 4GB system, which means conservative image size, conservative batch size, and a lean training loop instead of a paper-scale reproduction.

Published and testable

The project is in GitHub, the environment is installed, the checkpoint loads, and the dataset preparation path is scripted rather than manual.

Baseline commands

cd /home/alexander/depth-project
source .venv/bin/activate
python src/prepare_nyu_v2.py \
  --root /home/alexander/depth-project/data/nyu_v2 \
  --val-count 512

python src/train.py --config configs/baseline.yaml
python src/eval.py --config configs/baseline.yaml \
  --checkpoint checkpoints/baseline_best.pt

python src/plots.py \
  --metrics-log outputs/metrics/baseline_history.jsonl \
  --per-image outputs/metrics/baseline_test_per_image.csv \
