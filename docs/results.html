Paper story preserved

The project keeps the original presentation’s main structure: relative depth reasoning, Depth Anything Small, NYU-v2 evaluation, and an optional pseudo-label self-training extension.

Built for this machine

The implementation is sized for an RTX 3050 4GB system, which means conservative image size, conservative batch size, and a lean training loop instead of a paper-scale reproduction.

Published and testable

The project is in GitHub, the environment is installed, the checkpoint loads, and the dataset preparation path is scripted rather than manual.

Main conclusion

The fine-tuned checkpoint performed worse than the zero-shot baseline on the held-out test subset. That makes this run a valid proof-of-concept and a useful hyperparameter-analysis artifact, not just a success-only demo.

  • validation still improved across epochs
  • test performance dropped after fine-tuning
  • the result suggests overfitting on a small training subset
  • all 59 before/after image panels were saved for inspection
