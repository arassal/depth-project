Depth Anything • Results

Teacher, student, pseudo labels, and measured outcomes.

This project now follows the original paper story more closely: keep the Depth Anything Small architecture fixed, start from a strong teacher, expand the dataset with pseudo labels, and train a student on more data than the initial labeled seed set.

As of May 7, 2026, the teacher-student run is complete. It improved over the tiny supervised-only student but still did not beat the pretrained zero-shot teacher.

Architecture stayed fixed

The project did not change the core Depth Anything Small architecture. The main change was data scale through pseudo-label expansion.

Paper story matched better

The repo now explicitly shows teacher -> pseudo labels -> student retraining instead of stopping at a tiny supervised fine-tune.

Measured, not invented

All numbers on this page come from completed local runs on the held-out NYU-v2 subset test split.

Main conclusion

  • The zero-shot teacher remains the strongest overall model
  • The teacher-student run improved over the supervised-only student
  • The extra `2000` pseudo-labeled indoor images closed part of the gap
  • The local student still trails the pretrained teacher on held-out test data

Data growth and pseudo-label expansion

How to read the model outputs

Each NYU-v2 panel is organized as RGB | GT | Pred | Error. The key question is whether the model preserves large scene structure and relative depth ordering. The system should be interpreted as relative depth estimation, not exact metric distance measurement from one image.

Exact teacher-student workflow

cd /home/alexander/depth-project
source .venv/bin/activate
python src/prepare_indoor_pseudo_pool.py \
  --root data/unlabeled/indoor_teacher_pool \
  --limit 2000

python src/pseudo_label.py --config configs/teacher_student_poc.yaml
python src/train.py --config configs/teacher_student_poc.yaml
python src/eval.py --config configs/teacher_student_poc.yaml \
  --checkpoint checkpoints/teacher_student_poc_best.pt

Why this is the honest claim

  • The architecture stayed fixed
  • The dataset grew through pseudo labeling
  • The student improved over the tiny supervised baseline
  • The pretrained teacher remained stronger on the held-out test set
  • This is a valid proof of concept for the original teacher-student idea at local scale
